# Privacy Policy

DisplayFill does not collect, track, sell, or share personal data.

The app runs locally on your Mac and uses display information only to show and
adjust fill-light overlays on connected screens. Display settings and lighting
preferences are stored locally using macOS user defaults.

If camera-aware auto light is enabled, DisplayFill checks local system camera
running state so it can turn the light on while another app is using a camera.
DisplayFill does not capture, record, transmit, or inspect camera frames.

DisplayFill does not include analytics, advertising SDKs, account sign-in,
network services, or third-party tracking.

If you need support, open an issue at:

https://github.com/dashhuang/DisplayFill/issues
